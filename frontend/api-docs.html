Staff Management System

Staff API Docs

API Documentation

Base URL: http://localhost:5000/api

GET /staff

Retrieve all staff members from the system.

Response (200 OK):
{ "success": true, "message": "Staff members retrieved successfully", "data": [ { "id": 1, "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "position": "Software Engineer", "department": "IT", "salary": 75000, "joinDate": "2023-01-15" } ] }

GET /staff/:id

Retrieve a specific staff member by ID.

Parameter Type Description
id integer Staff member ID
Response (200 OK):
{ "success": true, "message": "Staff member retrieved successfully", "data": { "id": 1, "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "position": "Software Engineer", "department": "IT", "salary": 75000, "joinDate": "2023-01-15" } }

POST /staff

Create a new staff member.

Request Body:
{ "firstName": "Jane", "lastName": "Smith", "email": "jane.smith@example.com", "position": "Product Manager", "department": "IT", "salary": 85000, "joinDate": "2024-01-10" }
Response (201 Created):
{ "success": true, "message": "Staff member created successfully", "data": { "id": 2, "firstName": "Jane", "lastName": "Smith", "email": "jane.smith@example.com", "position": "Product Manager", "department": "IT", "salary": 85000, "joinDate": "2024-01-10" } }

PUT /staff/:id

Update an existing staff member.

Request Body:
{ "firstName": "Jane", "lastName": "Smith", "email": "jane.smith@example.com", "position": "Senior Product Manager", "department": "IT", "salary": 95000 }
Response (200 OK):
{ "success": true, "message": "Staff member updated successfully", "data": { "id": 2, "firstName": "Jane", "lastName": "Smith", "email": "jane.smith@example.com", "position": "Senior Product Manager", "department": "IT", "salary": 95000 } }

DELETE /staff/:id

Delete a staff member.

Response (200 OK):
{ "success": true, "message": "Staff member deleted successfully", "data": null }

Error Responses

When an error occurs, the API returns an error response with the appropriate status code:

400 Bad Request:
{ "success": false, "message": "Validation failed", "errors": [ "First name is required", "Valid email is required" ] }
404 Not Found:
{ "success": false, "message": "Staff member not found" }
500 Internal Server Error:
{ "success": false, "message": "Internal server error" }
